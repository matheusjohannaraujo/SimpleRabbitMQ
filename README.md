### A biblioteca <i>SimpleRabbitMQ</i> foi construída em PHP para simplificar o uso de Pub/Sub do RabbitMQ.

#### Compatível com PHP 7 e versões superiores

### Lista dos principais comandos:

> docker compose up -d

> composer update

> php pub.php

> php sub.php
